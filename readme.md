# Windows Dotfiles
## Prerequisites
1. Neovim [RipGrep]
1. Glazewm [Zebar]
1. PowerToys (PowerToys Run & ZoomIt)
1. Windows Terminal
<br>
Each folder has a `readme.md` on how to properly configure

## PowerToys Configuration
### ZoomIt
Change all keys to `ctrl+shift+` so if it was `ctrl+2`, it would be `ctrl+shift+2`
